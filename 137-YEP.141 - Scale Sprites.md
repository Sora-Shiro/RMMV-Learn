# 脚本功能概述

本脚本可以自动缩放精灵至 48x48 大小。

# 脚本可设置参数

- Smoothing，是否平滑缩放精灵

# Instructions 指南

将精灵文件命名为 [48x48] 即可被该脚本自动缩放
