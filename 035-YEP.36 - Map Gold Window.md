# 脚本功能概述

本脚本可以在行走界面添加金币窗口。

# 脚本可设置参数

- Automatic Open，是否自动在地图窗口打开金币窗口
- Default Position，窗口默认显示的位置
- Opacity，窗口的透明度

# Plugin Commands 脚本指令表

脚本指令|功能
:-|:-
OpenMapGoldWindow            |打开地图金币窗口
CloseMapGoldWindow            |关闭地图金币窗口
MapGoldWindowPosition x|将金币窗口的位置调整为 x *[1]*

\[1]: 可选 1-9 值，分别代表下列位置：
```
7  8  9
4  5  6
1  2  3
```
