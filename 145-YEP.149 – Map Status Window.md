# 脚本功能概述

本脚本可以在行走于地图时添加状态窗口。

# 脚本可设置参数

- Window X...
- Window Y...

# Plugin Commands 脚本指令表

脚本指令|功能
:-|:-
OpenMapStatusWindow            |打开窗口
CloseMapStatusWindow           |关闭窗口
ToggleMapStatusWindow          |关闭后再打开窗口再刷新
RefreshMapStatusWindow         |刷新窗口
SetMapStatusWindowX n|参考参数说明
SetMapStatusWindowY n|参考参数说明
