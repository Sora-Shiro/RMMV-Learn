# 脚本功能概述

本脚本可以添加帮助界面。

# 脚本可设置参数

1. General

- Help File Path，帮助文件路径
- Enable F1 Key，是否启用按下 F1 打开帮助文件

2. Menu

- Help Command...
- Auto Add Menu...
- Show Command...
- Auto Place Command...

# Plugin Commands 脚本指令表

脚本指令|功能
:-|:-
OpenHelp            |打开帮助界面
ShowMenuHelpCommand            |显示“Help”按钮
HideMenuHelpCommand          |显示“Help”按钮
