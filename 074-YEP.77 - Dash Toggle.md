# 脚本功能概述

本脚本可以改变冲刺是否可用。

# Notetags 备注信息表

备注信息|可作用范围|功能
:-|:-|:-
&lt;Disable Dashing>|角色、职业、武器、防具、状态|队伍的第一名队员如果被这个标签影响，那么玩家无法使用冲刺


# Plugin Commands 脚本指令表

脚本指令|功能
:-|:-
EnableDashing            |使冲刺可用
DisableDashing            |使冲刺不可用
ToggleDashing          |切换冲刺的可用状态
