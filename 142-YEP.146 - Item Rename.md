# 脚本功能概述

需要 `YEP_ItemCore.js` 插件支持。

本脚本可以更改物品的名字。

# 脚本可设置参数

- Rename Command...
- Default Rename，是否默认所有道具都能重命名
- Max Length，名字最大长度
- Rename Text...
