# 脚本功能概述

需要 `YEP_ItemCore.js` 和 `YEP_EquipCore.js` 插件支持。

本脚本改善了调整装备的方式。

之前在物品菜单调整装备（而不是装备菜单），现在可以通过装备菜单的 Customize 快捷跳转到物品菜单来调整装备，更符合直觉。
