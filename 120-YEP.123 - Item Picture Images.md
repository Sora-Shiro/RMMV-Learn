# 脚本功能概述

需要 `YEP_ItemCore.js` 插件支持。

本脚本可以为物品添加自定义图片说明。

# 脚本可设置参数

- Max Image Width，图片宽度
- Max Image Height，图片高度

# Notetags 备注信息表

备注信息|可作用范围|功能
:-|:-|:-
&lt;Picture: filename>|物品、武器、防具|设置图片，色调自动设为 0
&lt;Picture Image: filename>|物品、武器、防具|设置图片，不设置色调
&lt;Picture Hue: x>|物品、武器、防具|设置色调
