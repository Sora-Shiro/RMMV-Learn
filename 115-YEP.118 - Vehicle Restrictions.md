# 脚本功能概述

需要 `YEP_RegionRestrictions.js` 插件支持。

本脚本可以通过区域限制，限制交通工具的通行范围。

# 脚本可设置参数

1. Boat

- Boat Restrict，小船的限制区域 ID
- Boat Allow，小船的可行区域 ID
- Boat Land，小船可停靠区域 ID
