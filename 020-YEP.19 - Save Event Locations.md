# 脚本功能概述

本脚本可以存储事件的位置，这样玩家重新进入该地图时它们不会被重置，而是读取上一次的位置。

# Notetags 备注信息表

备注信息|可作用范围|功能
:-:|:-:|:-:
&lt;Save Event Locations>|地图|存储地图所有事件的位置
&lt;Save Event Location>|事件|存储该事件的位置

# Plugin Commands 脚本指令表

`ResetAllEventLocations` 重置所有事件的位置
