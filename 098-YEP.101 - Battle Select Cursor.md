# 脚本功能概述

本脚本可以添加目标选择光标。

# 脚本可设置参数

- Ally Cursor，选择我方时光标图片
- Enemy Cursor，选择敌方时光标图片
- Anchor X...
- Anchor Y...
- Position X...
- Position Y...
