# 脚本功能概述

本脚本可以添加读取游戏、战斗胜利或失败、关闭菜单、搭乘/离开载具时触发相应公共事件的功能。

# 脚本可设置参数

1. General

- Load Game Event，读取游戏后触发的公共事件
- Battle Won Event...
- Escape Battle Event...
- Close Menu Event...

2. Vehicles

- Boat Enter Event
- Boat Exit Event
- Ship Enter Event
- Ship Exit Event
- Airship Enter Event
- Airship Exit Event
