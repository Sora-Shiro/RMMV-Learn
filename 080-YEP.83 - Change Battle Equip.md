# 脚本功能概述

需要 `YEP_BattleEngineCore.js` 和 `YEP_EquipCore.js` 插件支持。

本脚本可以添加战斗时更换装备的功能。

# 脚本可设置参数

- Command Name，代替显示“Equip”的文本
- Equip Cooldown，冷却回合数
