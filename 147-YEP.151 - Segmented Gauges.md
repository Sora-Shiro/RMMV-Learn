# 脚本功能概述

本脚本可以将 HP/MP/TP/EXP/属性 条分割为多个定值小条。

# 脚本可设置参数

1. Segments

- HP Gauge Segments
- MP Gauge Segments
- TP Gauge Segments

2. Status Menu Core

- EXP Gauge Segments
- Param Gauge Segments
