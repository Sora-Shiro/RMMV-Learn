# 脚本功能概述

本脚本可以添加更多能显示在游戏菜单上的数据。

# 脚本可设置参数

1. Settings

- Window X...
- Window Y...
- Window Width...
- Window Height...
- Font Size...
- Standard Padding...
- Text Padding...
- Back Opacity...
- Window Opacity...

2. Variables

- Displayed Variable 1，变量 1 关联的全局变量
- Show Switch Variable 1，决定是否显示变量 1 的开关
- ...

（共有 10 个这样的 Displayed Variable/Show Switch Variable）
