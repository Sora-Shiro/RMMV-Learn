# 脚本功能概述

本脚本可以在地图上生成装备列表窗口。

# 脚本可设置参数

- Default Columns，默认列数
- Default Rows，默认行数
- Default X Position，默认 X 轴位置
- Default Y Position，默认 Y 轴位置
- Default Width，默认宽度
- Default Enable，默认使所有列出的技能可选
- Default Quantity，默认显示数量
